void com.griddynamics.qa.framework.CustomPerStoryWebDriverSteps.beforeStory ( ) throws InterruptedException

Action(s) before running each story:

  1. Initialize new instance of WebDriver;
  2. Clean browser cookies;
  3. Set page load timeout for most browsers, according to value of com.griddynamics.qa.framework.CustomPerStoryWebDriverSteps#WAIT_PAGE_OPEN_TIMEOUT_IN_MS property;
  4. Memorize window handle of the main window.
